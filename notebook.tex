    \section{k-Nearest Neighbors Classifier
(k-NN)}\label{k-nearest-neighbors-classifier-k-nn}

The k-NN is one of the simplest classifiers in Machine Learning.
Differently from other common supervised techniques, it does not perform
a \textbf{learning}; instead, the algorithm checks the distance between
the instance that will be classified and other feature vectors from the
dataset. Due to its simplicity, it is often used in benchmarks of
complex classifiers, like Artificial Neural Network (\textbf{ANN}) and
Suport Vector Machine (\textbf{SVM}).

This notebook shows how the dimensionality reduction algorithms required
for this work affects the accuracy of the k-NN classifier in the
classification task posed by the Covertype dataset. Sections are
organized as follows: first, we import all necessary libraries to run
k-NN from the \texttt{sklearn} library and load the dataset, separating
data from targets; then we load the results datasets and define a
function to perform k-NN over them, running also cross-validation to
validate the results; finally we compare the performance of the
classifier according to each reduced or extracted set of attributes.
